## Overview
This project analyses Uber trip data to understand booking trends, revenue patterns,
and trip efficiency using Power BI.

## Objectives
- Analyse total bookings and revenue
- Identify peak demand periods
- Evaluate trip distance and duration
- Understand vehicle and location-based performance

## Tools Used
- Power BI Desktop
- Power Query
- DAX

## Key KPIs
- Total Bookings
- Total Booking Value
- Average Booking Value
- Total & Average Trip Distance
- Average Trip Time

## Dashboards
- **Overview Analysis:** Bookings and revenue trends with dynamic KPIs  
- **Time Analysis:** Demand patterns by hour, day, and time intervals  
- **Details Analysis:** Location-wise and vehicle-wise trip insights

## Features
- Dynamic measure selection
- Interactive slicers and drill-through
- Vehicle and location analysis
